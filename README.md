Time Utility Web App

A simple web-based Time Utility app featuring a digital clock, stopwatch, and alarm clock.
Built with HTML, CSS, JavaScript, and Bootstrap 5.

------------------------------------------------------------

DEMO 

------------------------------------------------------------

FEATURES

Digital Clock
- Shows current time in HH:MM:SS AM/PM format
- Displays current date (optional)
- Can toggle digital font

Stopwatch
- Start, Stop, Resume, Reset functions
- Updates in real-time with milliseconds
- Displays current date

Alarm Clock
- Set a custom alarm time (e.g., 07:30 PM)
- Alerts when alarm time is reached

Settings
- Dark Mode
- Digital Font
- Hide Date

------------------------------------------------------------

INSTALLATION / USAGE

1. Clone the repository:
   git clone https://github.com/your-username/time-utility.git
2. Open index.html in a browser
3. Use the Settings button to toggle Dark Mode, Digital Font, or Hide Date
4. Set an alarm via the Set Alarm button

CUSTOMIZATION

- Change the time zone in script.js (default: Asia/Karachi)
- Change the font in style.css
- Modify colors or card styles in style.css
