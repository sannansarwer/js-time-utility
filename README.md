Time Utility Web App
A responsive web-based time utility that combines a digital clock, stopwatch, and alarm clock in a single interface. Includes customizable settings for dark mode, digital font, and date visibility.

Features
Digital Clock:
•	Displays the current time and date (default: Pakistan - Lahore).
•	Updates every second in real time.
Stopwatch:
•	High-precision stopwatch with Start, Stop, Resume, and Reset.
•	Displays minutes, seconds, and milliseconds.
•	Shows the current date.
Alarm Clock:
•	Set alarms in HH:MM AM/PM format.
•	Browser alert triggers when alarm time is reached.
•	Displays current time and date.
Settings:
•	Dark Mode – Night-friendly theme.
•	Digital Font – Futuristic digital clock font.
•	Hide Date – Hide dates on all clocks.

1.	Clone or download the repository.
2.	Navigate to the project folder.
3.	Open index.html in your web browser.
________________________________________
Usage
Digital Clock: Shows current time and date automatically.
Stopwatch:
•	Start to begin timing.
•	Stop to pause.
•	Resume to continue.
•	Reset to clear the stopwatch.
Alarm Clock:
•	Click Set Alarm and enter a time (e.g., 07:30 PM).
•	Browser alert will notify when the time is reached.
Settings:
•	Toggle Dark Mode for night-friendly display.
•	Enable Digital Font for a futuristic clock appearance.
•	Use Hide Date to remove date display from all clocks.
